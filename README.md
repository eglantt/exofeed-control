# ExoFeed Control

Приложение для учёта кормления и линьки животных в террариумном хозяйстве. Работает полностью в браузере, без сервера и без интернета (кроме самой первой загрузки страницы) — вся база хранится локально в браузере устройства (`localStorage`).

Текущая версия: **v1.5.5-alpha**.

**Важно:** сама база животных никуда на GitHub не отправляется и не хранится там — она остаётся только в браузере того устройства, с которого вы её вносите. Открыв сайт с двух разных устройств (телефон и планшет, например), вы получите две независимые базы — для переноса данных между ними по-прежнему нужен экспорт/импорт (кнопки «💾 Бэкап» и «📂 Импорт» внутри приложения).


## Установка как приложение (PWA) на телефон

После размещения на GitHub Pages сайт можно установить на главный экран как обычное приложение — со своей иконкой, без адресной строки браузера, и с работой без интернета после первого открытия.

**Android (Chrome):** откройте ссылку → в меню браузера (⋮) выберите **«Установить приложение»** (или Chrome сам покажет баннер с таким предложением).

**iPhone/iPad (Safari):** откройте ссылку → нажмите кнопку «Поделиться» (квадрат со стрелкой вверх) → **«На экран «Домой»»**.

После установки на экране появится иконка ExoFeed — открывается как отдельное приложение. Данные (база животных) при этом точно такие же, как в браузере — это не отдельное хранилище, а тот же `localStorage` того же сайта.
